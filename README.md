# Gen Z Career Aspirations Analysis
## Independent Project - KultureHire Internship

## Project Objective:
This project aims to analyze the career aspirations and challenges faced by GenZ, alongside the retention struggles encountered by employers. By examining various factors such as job preferences, expectations, and career goals of GenZ, we identified key areas influencing their career choices. Additionally, the project explored the hurdles employers face in retaining GenZ employees, including work-life balance, growth opportunities, and job satisfaction. The insights derived from this analysis help bridge the gap between GenZâ€™s expectations and employer strategies, providing actionable recommendations for improving retention and fostering a motivated workforce.
